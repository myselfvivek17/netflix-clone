# Netflix Clone 🎥

A front-end clone of the Netflix landing page, built with HTML and CSS.

## About

This project replicates the Netflix homepage UI, including the hero section, feature highlights, and FAQ layout — great for practising responsive web design.

## Features

- Netflix-style hero section
- Feature highlights section
- FAQ accordion layout
- Responsive design

## Tech Stack

- HTML5
- CSS3

## Getting Started

```bash
git clone https://github.com/myselfvivek17/netflix-clone.git
cd netflix-clone
# Open index.html in your browser
```
